Rules

Hack Rules

75% of you told us you want to hack all night with your colleagues and have fun, so we are opening up the Grabathon. Your Grabathon submissions needn't be solving a problem for Grab.

Only 3-5 members are allowed in the same team. You are only allowed to be part of one participating team.

Please register your teams before 5th May 2017 via this Google Form.

Hardware entries are very welcome! Do note that you have to procure the hardware yourself before the hackathon. The budget for the hardware needs to be pre-approved by your line manager.

Please help keep our spaces clean and neat throughout the Grabathon.

Each team is given up to 3 minutes to present.

Developer Rules

Do not make any modifications on staging or production, or teams will face disqualification.

Do not use existing staging / production keys if you are using 3rd party tools like Twilio / Nexmo / Google etc.

Existing public API on staging is accessible for your projects.

Driver API is available via Driver Cendol API.
Passenger API is available via Passenger V2/V3 API.
Alternatively, you might want to check out Marionette and Marionette API by the Internal Tools team.
